    Msc Internship

    Fitting a psychometrics curve using pyTorch

    Under the supervision of Laurent Perrinet

  • Methods : In psychophysics, subject responses constitute binary data. The researcher goal is to find the distribution that fits the best to our binary data. However, fitting the curve can be a difficult task. Participant can make carelessness errors that can biased the shape of the psychometric curve To limit this bias, we did a preliminary study with synthetic data and define a fittting method using pyTorch
